### Html table structure

```html
<table>
  <tr>
    <th>Repo</th>
    <th>Web</th>
    <th>Stars</th>
  </tr>
  <tr>
    <td><a href="<Repo>">Name</a></td>
    <td><a href="<WebSite>">ğŸŒ</a></td>
    <td>
      <a href="<Repo/stargazers>">
         <img src="https://img.shields.io/github/stars/<User>/<Repo>?style=plastic">
      </a>
    </td>
  </tr>
</table>
```
